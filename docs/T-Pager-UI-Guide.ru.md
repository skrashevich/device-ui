# Полное руководство T-Pager UI

Полное практическое руководство по использованию интерфейса T-Pager Meshtastic с физической клавиатурой и энкодером.

---

## 1. Введение

T-Pager — компактное портативное устройство для сетевой коммуникации Meshtastic с 480×222 пиксельным дисплеем и интуитивным интерфейсом управления.

### Аппаратные элементы управления

**Ротационный энкодер (Rotary Encoder)**
- Вращение: навигация по элементам интерфейса (вверх/вниз)
- Нажатие: подтверждение выбора (эквивалент Enter)
- Быстрое вращение: автоматическое ускорение навигации (4x при <50 мс между шагами)

**Физическая клавиатура TCA8418**
- Матрица 4 ряда × 10 колонок
- Модификаторы: Shift, Sym
- Полная поддержка текстового ввода и навигации

### Модификаторы и специальные функции

**Shift** (липкий переключатель — sticky toggle)
- Одно нажатие: активирует режим Shift
- Следующий нажатый символ будет прописным
- Автоматически деактивируется после первого символа

**Sym** (липкий переключатель)
- Одно нажатие: активирует режим Sym (специальные символы и команды)
- Следующий нажатый символ выдаст спецсимвол или навигационную команду
- Автоматически деактивируется после первого символа

## 3. Энкодер и его управление

### Основные движения энкодера

| Действие | Результат |
|----------|-----------|
| Вращение вверх (против часовой) | Навигация к предыдущему элементу / приближение (на карте) |
| Вращение вниз (по часовой) | Навигация к следующему элементу / отдаление (на карте) |
| Нажатие кнопки энкодера | Подтверждение выбора (Enter) |

### Специальные режимы энкодера

**Прокрутка содержимого (Alt/Sym режим)**
- Держите клавишу **Sym**, одновременно вращая энкодер
- Содержимое панели прокручивается, но фокус остаётся на текущем элементе
- Полезно для просмотра длинных списков без потери позиции ввода

**Ускорение навигации**
- Быстрое вращение энкодера автоматически увеличивает шаг навигации:
  - < 50 мс между шагами: **4x** ускорение
  - < 100 мс между шагами: **2x** ускорение
  - ≥ 100 мс между шагами: обычная скорость

## 4. Навигация между экранами

### Главные экраны T-Pager

T-Pager имеет **6 основных экранов** доступных из главного экрана (Home):

1. **Home** — главный экран с быстрыми кнопками и индикаторами
2. **Nodes** — список известных узлов сети
3. **Groups** — список групп/каналов
4. **Messages** — просмотр сообщений и чатов
5. **Map** — интерактивная карта с позициями узлов
6. **Settings** — настройки устройства и сети

### Способы переключения между экранами

| Способ | Действие |
|--------|---------|
| **Sym+E** | Переключиться на следующий экран (цикл) |
| **Sym+F** | Переключиться на предыдущий экран |
| **Sym+Q** | Перейти сразу на Home экран |
| **Энкодер** (в боковом меню) | Прокрутка по кнопкам экранов |
| **Нажатие на кнопку** | Перейти на соответствующий экран |

### Фокус после переключения

Когда вы переходите на новый экран, фокус автоматически устанавливается на первый интерактивный элемент:

| Экран | Первый элемент |
|-------|----------------|
| Home | Кнопка Messages |
| Nodes | Первый узел в списке |
| Groups | Первый канал |
| Messages | Поле ввода текста |
| Map | Кнопка навигации |
| Settings | Первый параметр |

## 5. Экран Главная (Home)

### Элементы главного экрана

Главный экран содержит быстрые кнопки доступа и индикаторы состояния:

**Основные кнопки** (навигация энкодером):
- **Messages** — быстрый переход к сообщениям
- **Nodes** — просмотр списка узлов
- **Groups** — управление каналами и группами
- **Map** — просмотр карты с позициями узлов
- **Settings** — конфигурация устройства

**Индикаторы состояния** (правый верхний угол):
- Батарея (уровень заряда)
- Сигнал LoRa (качество сигнала)
- Время
- Статус GPS
- WiFi / MQTT / Bluetooth соединение
- Статус SD-карты
- Объём памяти

### Взаимодействие с элементами

| Действие | Результат |
|----------|-----------|
| Энкодер вверх/вниз | Навигация по кнопкам |
| Enter / Нажатие энкодера | Активировать выбранную кнопку |
| Sym+Q / Backspace (вне текстового поля) | Вернуться на Home |

## 6. Экран Узлы (Nodes)

### Список узлов сети

Этот экран отображает все известные узлы в сети Meshtastic.

**Способы входа:**
- Нажать кнопку "Nodes" на главном экране
- Использовать Sym+E для циклического переключения между экранами

### Навигация в списке узлов

| Действие | Результат |
|----------|-----------|
| Энкодер вверх/вниз | Прокрутка по узлам |
| Быстрое вращение энкодера | Ускоренная навигация (jump by 5) |
| Enter / Нажатие энкодера | Выбрать узел и показать детали |
| Backspace / Sym+R | Вернуться на Home |

### Информация о каждом узле

Для каждого узла отображается:
- **Имя** — идентификатор узла
- **Статус** — онлайн/офлайн
- **Последний контакт** — когда в последний раз был контакт с узлом
- **Позиция** — если доступна GPS информация

**Фокус на узле подсвечивает его на карте** (если карта доступна).

### Детали узла

После выбора узла можно:
- Отправить сообщение узлу
- Просмотреть метрики и информацию
- Отслеживать историю контактов

## 7. Экран Группы и Каналы (Groups)

### Список групп и каналов

Этот экран показывает все доступные группы и каналы для коммуникации.

**Способы входа:**
- Нажать кнопку "Groups" на главном экране
- Использовать Sym+E для переключения между экранами

### Навигация по группам

| Действие | Результат |
|----------|-----------|
| Энкодер вверх/вниз | Прокрутка по группам/каналам |
| Enter / Нажатие энкодера | Открыть чат группы |
| Backspace / Sym+R | Вернуться на Home |

### Информация о группе

Для каждой группы показывается:
- **Имя канала** — название группы
- **Номер канала** — идентификатор
- **Непрочитанные сообщения** — индикатор (сердце ❤ указывает на новые сообщения)
- **Активность** — визуальный индикатор последней активности

## 8. Экран Сообщения (Messages)

### Просмотр сообщений и чатов

На этом экране можно просматривать сообщения из узлов и групп, а также писать новые сообщения.

**Способы входа:**
- Нажать кнопку "Messages" на главном экране
- Нажать конверт на главном экране
- Выбрать узел или группу из других экранов

### Структура экрана сообщений

**Верхняя часть — список чатов:**
- Энкодер: навигация по чатам
- Enter: открыть чат

**Нижняя часть — поле ввода и история:**
- История сообщений в выбранном чате
- Поле ввода текста сообщения

### Ввод текста (в поле сообщения)

| Действие | Результат |
|----------|-----------|
| Обычные клавиши | Печать символов |
| Shift+символ | Заглавная буква |
| Sym+символ | Специальный символ |
| Enter | Отправить сообщение |
| Backspace | Удалить символ перед курсором |
| Энкодер вверх/вниз | Перемещение курсора в тексте |
| Sym+энкодер | Прокрутка истории сообщений |

### Отправка сообщения

1. Наберите текст сообщения в поле ввода
2. Нажмите **Enter** для отправки
3. Сообщение будет отправлено и поле очистится

**Специальная функция:** При пустом поле ввода, энкодер прокручивает историю сообщений вверх/вниз для просмотра старых сообщений.

### Навигация в истории сообщений

| Действие | Результат |
|----------|-----------|
| Энкодер (пустое поле) | Прокрутка истории вверх/вниз |
| Sym+энкодер (с текстом в поле) | Прокрутка несмотря на текст |
| Sym+W (Sym+Up) | Прокрутка вверх (без фокуса на поле) |
| Sym+S (Sym+Down) | Прокрутка вниз (без фокуса на поле) |

## 9. Экран Карта (Map)

### Интерактивная карта позиций узлов

Карта отображает все известные узлы на интерактивной географической карте.

**Способы входа:**
- Нажать кнопку "Map" на главном экране
- Нажать кнопку Location на главном экране (центрирует на текущей позиции)

### Панорамирование карты (Sym+WASD)

| Комбинация | Действие |
|-----------|---------|
| **Sym+W** | Панорамирование вверх |
| **Sym+S** | Панорамирование вниз |
| **Sym+A** | Панорамирование влево |
| **Sym+D** | Панорамирование вправо |

### Зум (масштабирование)

| Способ | Действие |
|--------|---------|
| Энкодер вверх | Приближение (zoom in) |
| Энкодер вниз | Отдаление (zoom out) |
| Enter (энкодер) | Приближение на один уровень |

### Элементы управления картой

**Верхние кнопки:**
- **Home** — центрировать карту на текущей позиции
- **Zoom In (+)** — приближение
- **Zoom Out (-)** — отдаление
- **GPS Lock** — привязать карту к текущей позиции (следование GPS)

**Слайдеры:**
- **Яркость** — яркость отображения карты
- **Контрастность** — контраст карты

**Выбор провайдера:**
- **OSM (OpenStreetMap)** — открытая карта
- **Yandex** — альтернативный провайдер

### Взаимодействие с узлами на карте

| Действие | Результат |
|----------|-----------|
| Энкодер на узел | Выбор узла |
| Enter (узел выбран) | Открыть детали узла |
| Нажатие на узел (если есть сенсор) | Открыть чат с узлом |

### Навигация на карте

| Действие | Результат |
|----------|-----------|
| Sym+Q | Вернуться на Home |
| Sym+R | Закрыть карту |
| Sym+E | Следующий экран |
| Backspace | На Home экран |

## 10. Экран Настройки (Settings)

### Конфигурация устройства

На этом экране настраиваются все параметры T-Pager.

**Способы входа:**
- Нажать кнопку "Settings" на главном экране
- Использовать Sym+E для циклического переключения

### Основные категории настроек

**Устройство (Device Settings):**
- Имя устройства
- Роль узла (Client, Router, Repeater, Tracker и т.д.)
- Регион (US, EU_433, EU_868, RU, и т.д.)

**LoRa Settings:**
- Модемный профиль (Long Fast, Medium Slow, Short Turbo и т.д.)
- Номер канала
- Мощность передачи

**Display Settings:**
- Яркость экрана
- Timeout экрана (время перед сном)
- Блокировка экрана (PIN-код)
- Тема оформления (тёмная/светлая)

**Сетевые (Network Settings):**
- WiFi соединение
- MQTT параметры
- Bluetooth опции

**Управление вводом (Input Control):**
- Сенсор движения
- Энкодер чувствительность

### Навигация в настройках

| Действие | Результат |
|----------|-----------|
| Энкодер вверх/вниз | Прокрутка по параметрам |
| Enter | Открыть параметр для редактирования |
| Backspace / Sym+R | Вернуться на Home (с подтверждением) |

### Работа с Dropdown'ами (выпадающие меню)

**Открытие и выбор:**
- Энкодер вверх/вниз: прокрутка опций в раскрытом меню
- Sym+W / Sym+S: переключение опций
- Enter: подтвердить выбор

**Поведение:**
- При фокусе на dropdown меню автоматически раскрывается
- Содержимое меню прокручивается при получении фокуса

### Работа с текстовыми полями

| Действие | Результат |
|----------|-----------|
| Обычные клавиши | Печать текста |
| Backspace | Удалить символ |
| Enter | Подтвердить и закрыть поле |
| Shift+символ | Заглавная буква |
| Sym+символ | Специальный символ |

### Работа со слайдерами

| Действие | Результат |
|----------|-----------|
| Энкодер влево | Уменьшить значение |
| Энкодер вправо | Увеличить значение |
| Enter | Подтвердить |

### OK/Cancel кнопки

Каждый диалог настроек имеет кнопки подтверждения:
- **OK** — сохранить изменения и закрыть
- **Cancel** — отменить изменения и закрыть

Переключение между ними: Tab или энкодер вверх/вниз.

## 11. Русская раскладка (RU Layout)

### Переключение на русский язык

**Способ переключения:**
1. Нажмите **Sym+Shift** одновременно (или быстро один за другим в окне 400 мс)
2. Раскладка переключится с EN на RU (или обратно)
3. На экране появится индикатор текущей раскладки

**Ограничение:** Переключение доступно только вне текстовых полей.

**Cooldown:** После переключения требуется подождать 700 мс перед следующим переключением.

### Русская раскладка ЙЦУКЕН

При включённой RU раскладке все символы выдают русские буквы.

**Нормальный слой (базовые буквы):**
```
й ц у к е н г ш щ з
ф ы в а п о л д ж э
я ч с м и т ь б ю
```

**Shift слой (заглавные буквы):**
```
Й Ц У К Е Н Г Ш Щ З
Ф Ы В А П О Л Д Ж Э
Я Ч С М И Т Ь Б Ю
```

### Доступ к дополнительным буквам (Sym слой)

Некоторые русские буквы недоступны в нормальном слое. Используйте Sym+key для их ввода:

| Комбинация | Буква | Описание |
|-----------|-------|---------|
| Sym+C | ж | жизнь |
| Sym+H | х | хорошо |
| Sym+J | э | этот |
| Sym+K | ъ | твёрдый знак |
| Sym+G | ё | ёлка |
| Sym+M | ю | юг |
| Sym+N | б | берег |

### Использование русского ввода

1. **Переключитесь на русскую раскладку:** Sym+Shift
2. **Откройте текстовое поле** (например, поле ввода сообщения)
3. **Печатайте русский текст** — все символы будут на кириллице
4. **Используйте Shift** для заглавных букв
5. **Используйте Sym** для дополнительных букв (см. таблицу выше)

**Раскладка сохраняется** между сессиями до следующего переключения.

## 12. Блокировка экрана и режим сна

### Экономия энергии

T-Pager автоматически переходит в режим сна после периода неактивности для сохранения батареи.

### Режим сна (экран выключен)

**Когда экран уходит в сон:**
1. Дисплей выключается (экран чёрный)
2. Энкодер и клавиатура остаются активными

**Пробуждение из сна:**
- Нажмите любую клавишу на клавиатуре
- Или нажмите энкодер (кнопка энкодера)
- Экран включится и вернёт вас к последнему экрану

### Блокировка экрана (PIN-код)

Если включена защита экрана, при пробуждении потребуется ввод PIN-кода.

**Ввод PIN-кода:**

| Действие | Результат |
|----------|-----------|
| Энкодер вверх/вниз | Выбор цифры (0-9) |
| Enter / Нажатие энкодера | Подтвердить цифру и перейти к следующей |
| Backspace | Удалить последнюю введённую цифру |
| Sym+R / ESC | Отменить ввод PIN |

**Отображение:**
- PIN отображается как точки для безопасности
- Каждая цифра вводится отдельно

### Настройка таймауа сна

В Settings → Display Settings:
- **Screen Timeout** — время в секундах до автоматического сна
- **Lock Screen** — включить/отключить PIN защиту

## 13. Советы и трюки

### Быстрая навигация

**Переключение между экранами:**
- Используйте **Sym+E** для циклического переключения между основными экранами
- Это быстрее, чем пользоваться энкодером в боковом меню

**Быстрый возврат на Home:**
- Нажмите **Backspace** вне текстового поля
- Или используйте **Sym+Q**
- Это работает с любого экрана

### Ускоренная навигация по спискам

**Быстрая прокрутка:**
- Вращайте энкодер быстро — скорость навигации автоматически увеличится
- 4x ускорение при очень быстром вращении (<50 мс между шагами)

**Page Jump в списке узлов:**
- Быстрое вращение энкодера прокручивает на 5 узлов за раз

### Эффективный ввод текста

**Переключение раскладки:**
- Переключитесь на русскую раскладку **один раз** перед написанием сообщения
- Раскладка сохранится до следующего переключения
- Экономит время при написании длинных сообщений на русском

**Специальные символы:**
- Используйте Sym+key для специальных символов
- Для русской раскладки используйте таблицу Sym слоя

### Прокрутка в сообщениях

**Чтение истории:**
- Очистите поле ввода (Backspace всё содержимое)
- Вращайте энкодер вверх для просмотра старых сообщений
- Это удобнее, чем использовать Sym+энкодер

**Прокрутка при письме:**
- Держите **Sym** и вращайте энкодер
- История будет прокручиваться, но фокус останется в поле ввода

### Работа с настройками

**Быстрый доступ:**
- Settings → Display Settings → Screen Timeout для управления энергией
- Settings → Device → Role для изменения роли узла

**Подтверждение изменений:**
- Нажмите **OK** для сохранения
- Нажмите **Cancel** для отмены (или Backspace с подтверждением)

### Эффективная работа с картой

**Панорамирование:**
- Используйте **Sym+WASD** для быстрого панорамирования
- Это намного удобнее, чем нажимать стрелки

**Масштабирование:**
- Энкодер вверх/вниз для zoom in/out
- Быстрое вращение ускоряет изменение масштаба

**GPS Lock:**
- Включите GPS Lock для следования текущей позиции
- Отключите для свободного панорамирования

### Фокусировка элементов

**Визуальный feedback:**
- Активный элемент подсвечивается (видна рамка или выделение)
- Индикатор Sym (оранжевый "S") показывает активность модификатора

**Переключение между элементами:**
- Используйте **Tab** для переключения на следующий элемент
- Используйте **Shift+Tab** для переключения на предыдущий элемент
- Это удобнее, чем энкодер, когда элементы расположены близко

### Работа с popups и диалогами

**Закрытие всплывающих окон:**
- Enter на кнопке: активировать действие
- Backspace: закрыть без действия
- Любая другая клавиша: просто закрыть

**Переключение между кнопками:**
- Энкодер вверх/вниз для выбора кнопки
- Sym+влево/вправо также работает

## Дополнительная информация

### Индикаторы на экране

- **Orange "S"** — активен модификатор Sym
- **RU/EN** — текущая раскладка клавиатуры
- **Батарея** — уровень заряда (%)
- **LoRa сигнал** — качество сигнала (1-5 полосок)
- **GPS** — статус спутникового позиционирования
- **WiFi/MQTT** — сетевые соединения

### Производительность

- Все операции выполняются локально на устройстве
- Карта и списки могут требовать времени при большом количестве узлов
- Используйте фильтры на экране Nodes для ускорения поиска

### Батарея и энергопотребление

- LoRa приём: основной потребитель энергии
- Установите больший Screen Timeout в Settings для экономии
- Отключите GPS, если не требуется отслеживание позиции
- Режим сна автоматически экономит энергию после заданного времени
